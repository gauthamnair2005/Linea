Why Choose Linea?

⚡ Blazing Fast

Compiled to native machine code with Rust-level performance. 100-1000x faster than Python interpreters.

🔒 Memory Safe

Automatic memory management with compile-time borrow checking. Zero use-after-free or buffer overflow bugs.

📦 Zero Deps

Standalone executable binaries. Only libc required at runtime. No dependency hell or version conflicts.

✨ Clean Syntax

Simple, readable syntax with type inference. No boilerplate, no surprises. Easy to learn and teach.

🎯 Type Safe

Static type checking catches errors at compile time. Instant feedback with helpful error messages.

🌍 Portable

Works on Linux, macOS, and Windows x86-64. Same binary runs everywhere without recompilation.

Quick Start

1. Install the Compiler

$ curl -fsSL https://github.com/gauthamnair2005/Linea/releases/download/v3.0.1/install.sh | sh
$ linea --help

2. Create a Program

$ cat > hello.ln << 'EOF'
var greeting = "Hello, Linea!"
display greeting
EOF

3. Compile and Run

$ linea compile hello.ln -o hello
$ ./hello
Hello, Linea!

That's it! Your executable runs anywhere. No Linea needed.

Language Features

Variables & Types

var x = 42
var pi = 3.14159
var name = "Linea"
var active = true

Control Flow

if x > 0 {
    display "positive"
} else {
    display "non-positive"
}

Loops

for i from 1~10 {
    display i
}

while x > 0 {
    x = x - 1
}

Arrays

var arr = [1, 2, 3, 4]
var mixed = [1, "two", 3.0]
display arr

Operators

var sum = a + b
var power = x ^ 2
var isEqual = x == y
var result = a && b || c

Type Casting

var count = "42"
typeCast num = int from count
display num + 8  // 50

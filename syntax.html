Linea Syntax

  • Printing a text: `display `
  • Displaying declared variables: `display @var`
  • Displaying declared actor variables: `display @act`
  • Displaying variable addresses: `display @adr`
  • Clearing the screen: `clrscr`
  • Displaying data types of declared variables: `display @dataType`
  • Displaying data type of a variable: `type `
  • Type casting a variable: `typecast `
  • Declaring a variable: `var = `
  • Updating a variable: `varupd = `
  • Declaring an array: `var[] = , , , ...`
  • Declaring a pointer: `pointer = `
  • Declaring an array of pointers: `pointer[] = , , , ...`
  • Updating a pointer: `pointerupd `
  • Taking input: `input `
  • Binary view: `bin `
  • Hexadecimal view: `hex `
  • Octal view: `oct `
  • Variable operations: `+, -, *, /, %, **, ^`
  • For loop: `for from ~ `
  • For loop (0 to specified): `for till `
  • For loop (1 to specified): `for noDuckTill `
  • Actors: `act = `
  • Killing actors: `killAct `
  • Killing variables: `kill `
  • Killing all variables: `killAll`
  • Killing all actors: `killActAll`
  • Plotting a graph: `plot `
  • Worker function definition: `worker = `
  • Worker function call: `workerCall `

Linea Syntax

            
    
                
  • display <string/variable> - Printing a text
    • 
                
  • display @var - Displaying declared variables
    • 
                
  • display @act - Displaying declared actor variables
    • 
                
  • display @adr - Displaying variable addresses
    • 
                
  • clrscr - Clearing the screen
    • 
                
  • display @dataType - Displaying data types of declared variables
    • 
                
  • type <variable> - Displaying data type of a variable
    • 
                
  • typecast <datatype> <variable> - Type casting a variable
    • 
                
  • var <variable> = <value> - Declaring a variable
    • 
                
  • varupd <variable> = <value> - Updating a variable
    • 
                
  • var[] <variable> = <value1>, <value2>, <value3>, ... - Declaring an array
    • 
                
  • pointer <variable> = <value> - Declaring a pointer
    • 
                
  • pointer[] <variable> = <value1>, <value2>, <value3>, ... - Declaring an array of pointers
    • 
                
  • pointerupd <variable> <value> - Updating a pointer
    • 
                
  • input <variable> <query> - Taking input
    • 
                
  • bin <value> - Binary view
    • 
                
  • hex <value> - Hexadecimal view
    • 
                
  • oct <value> - Octal view
    • 
                
  • +, -, *, /, %, **, ^ - Variable operations
    • 
                
  • for <iterator> from <start>~<end> <action> - For loop
    • 
                
  • for <iterator> till <times> <action> - For loop (0 to specified)
    • 
                
  • for <iterator> noDuckTill <times> <action> - For loop (1 to specified)
    • 
                
  • act <variable> = <value> - Actors
    • 
                
  • killAct <variable> - Killing actors
    • 
                
  • kill <variable> - Killing variables
    • 
                
  • killAll - Killing all variables
    • 
                
  • killActAll - Killing all actors
    • 
                
  • plot <variable> - Plotting a graph
    • 
                
  • worker <variable> = <expression> - Worker function definition
    • 
                
  • workerCall <variable> - Worker function call
    • 
                
  • file <filename> read - File reading

`blockchain`, `gpu_tools`, and `memory`

Proof-of-work + Merkle helpers, GPU vendor/iGPU discovery, and safe low/mid-level memory handles.

Blockchain

import blockchain

var root = blockchain::merkleRoot(["a", "b", "c"])
var mined = blockchain::mineBlock(1, "0000", root, 3)
display root
display mined

GPU Tools

import gpu_tools

display gpu_tools::adapters()
display gpu_tools::bestAdapter()
display gpu_tools::hasIGPU()
display gpu_tools::vendorName(0x10DE)

Memory

import memory

var h = memory::alloc(32)
memory::writeU8(h, 0, 255)
display memory::readU8(h, 0)
display memory::stats()
memory::free(h)

Linea

An AI-first, compiled programming language with native GPU acceleration, advanced reasoning algorithms, and beautiful syntax for machine learning.

Get Started View on GitHub

🚀 Key Features

Compiled & Fast

Compile Linea to standalone executables with Rust-level performance and automatic thread-aware parallel compilation jobs.

🎮 GPU Acceleration

Automatic GPU detection and optimization. Supports dGPU, iGPU, and CPU fallback with WGPU cross-platform compute.

🧠 Native ML/AI

Pure Linea machine learning libraries for training and inference. Neural networks, optimizers, and loss functions built-in.

🔬 ARL-Tangram Algorithm

Advanced reasoning with adaptive compositional learning. Learn complex reasoning patterns with multi-layer attention mechanisms.

🛡️ Memory Safe

Memory safety by design. No null pointers, no buffer overflows. Type-safe abstractions prevent entire classes of bugs.

📦 Rich Standard Library

HTTP, CSV, Excel, Graphics, Markdown support. Networking, data processing, and visualization out of the box.

🧩 Third-Party Package Manager

Install external `.ln` modules from an XML registry with automatic dependency installation into your local `libs/` folder.

Quick Start

Install Linea

git clone https://github.com/gauthamnair2005/Linea.git cd Linea ./install.sh

Your First Program

// hello.ln fn main() { var x @ int = 42 display x }

Compile & Run

linea compile hello.ln ./hello

Install Third-Party Library

python3 package-manager/linea_pkg.py greeter --registry ~/projects/linea-third-party-registry --libs-dir ./libs

📚 Example: Neural Network Training

// Neural network training with Linea import ml import datasets fn main() { // Load dataset var data = datasets::load_csv("iris.csv") var X = data::features var y = data::labels // Build model var model = ml::Sequential([ ml::Dense(4, 16), ml::ReLU(), ml::Dense(16, 8), ml::ReLU(), ml::Dense(8, 3) ]) // Train var optimizer = ml::Adam(learning_rate: 0.001) for epoch from 0~100 { var loss = model.train_step(X, y, optimizer) display "Epoch " + epoch + ": loss = " + loss } }

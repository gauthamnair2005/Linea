Literals

Literals are inline fixed values like numbers, strings, booleans, and list values.

var i @ int = 12
var f @ float = 3.5
var s @ str = "hello"
var b @ bool = False
var arr @ [int] = [10, 20, 30]

Literal-driven Defaults

var retries @ int = 3
var timeout_ms @ int = 2000
var mode @ str = "fast"

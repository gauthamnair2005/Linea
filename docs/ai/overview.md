# AI & Machine Learning in Linea

Linea treats AI as a first-class citizen.

## Features

- **Tensor Operations**: `+`, `-`, `*`, `/` on generic multi-dimensional arrays.
- **Autograd**: Automatic differentiation for training.
- **GPU Acceleration**: Seamless execution on NVIDIA, AMD, and Intel GPUs.

## Example

```linea
import ml
var model = ml::Linear(10, 2)
```

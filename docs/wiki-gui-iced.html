GUI Toolkit with Rust Iced

Linea v4.7.0 introduces a native GUI toolkit module built on Rust iced.

APIs

import gui

var ok = gui::window("Title", "Hello GUI", 640, 420)
display ok

var ok2 = gui::buttonWindow("Title", "Press the button", "Click me", 640, 420)
display ok2

Notes

GUI APIs are designed for compiled mode (`linea compile`). Interpreter mode reports a clear runtime error for GUI calls.

Third-Party Package Manager

Dependency-aware installation of external `.ln` libraries into local `libs/`.

Command

python3 package-manager/linea_pkg.py greeter \
  --registry ~/projects/linea-third-party-registry \
  --libs-dir ./libs

Registry Structure

linea-third-party-registry/
  libs/
    greeter.ln
  metadata/
    greeter.xml

Metadata XML

<package>
  <name>greeter</name>
  <version>1.0.0</version>
  <developer>Contributor Name</developer>
  <description>Simple greeting module</description>
  <entrypoint>libs/greeter.ln</entrypoint>
  <dependencies>
    <dependency name="textfmt" version=">=1.0.0" />
  </dependencies>
</package>

Behavior

The installer resolves transitive dependencies, prevents cyclic graphs, copies package files into `libs/`, and writes `libs/.linea-packages.lock.json`.

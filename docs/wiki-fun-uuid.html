`fun` and `uuid` Libraries

Small but useful helpers for randomization, playful outputs, and ID generation.

Fun APIs

import fun

display fun::coinFlip()
display fun::rollDice(6)
display fun::randomEmoji()
display fun::randomJoke()
display fun::randomColor()
display fun::choose(["ship", "iterate", "refactor"])

UUID APIs

import uuid

display uuid::v4()
display uuid::short()

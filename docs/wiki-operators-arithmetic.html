Arithmetic Operators

Use arithmetic operators for integer and float math in expressions and loops.

Supported Operators

  • + addition
  • - subtraction
  • * multiplication
  • / division
  • % remainder (integers)

Example: Basic Math

var a @ int = 10
var b @ int = 3

display a + b   // 13
display a - b   // 7
display a * b   // 30
display a / b   // 3
display a % b   // 1

Example: Running Total

var sum @ int = 0
for i from 1~6 {
    sum = sum + i
}
display sum

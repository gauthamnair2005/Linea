CSV Library

Use the csv module for tabular file read/write workflows.

import csv

var rows = csv::read("examples/iris_dummy.csv")
display rows

Write Example

import csv

var out_rows @ [str] = ["name,score", "alice,91", "bob,88"]
csv::write("tmp_scores.csv", out_rows)
display "csv written"

Tip

Validate column counts before transforming rows to avoid malformed downstream data.

Comprehensive guide to Modules in Linea

Importing Built-In Modules

Linea modules are imported once and used via namespace calls.

Example

import ml
import sql
import system

fn main() {
    var db = sql::open("app.db")
    var cwd = system::cwd()
    display cwd
    display db
}

Third-Party Modules via Package Manager

Install custom modules into `libs/` using the package manager in the main repository.

python3 package-manager/linea_pkg.py greeter \
  --registry ~/projects/linea-third-party-registry \
  --libs-dir ./libs

After install:

import greeter

fn main() {
    greeter::hello("Developer")
}

Registry Metadata Format

Each third-party module has metadata at `metadata/<libname>.xml` in the registry repository.

<package>
  <name>greeter</name>
  <version>1.0.0</version>
  <developer>Contributor Name</developer>
  <description>Greeter helper</description>
  <entrypoint>libs/greeter.ln</entrypoint>
  <dependencies>
    <dependency name="textfmt" version=">=1.0.0" />
  </dependencies>
</package>

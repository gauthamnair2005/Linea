Neural Network Layers

Comprehensive guide to Neural Network Layers in Linea

ML Layers

The ml module provides layer primitives for model composition.

import ml

var model = ml::Sequential([
    ml::Dense(4, 16),
    ml::ReLU(),
    ml::Dense(16, 3)
])

display model

Forward Pass Example

var x = [[5.1, 3.5, 1.4, 0.2]]
var logits = model.forward(x)
display logits

Logical Operators

Logical operators combine boolean expressions in guards and filters.

Core Operators

  • and
  • or
  • not
var age @ int = 22
var has_id @ bool = True

if age >= 18 and has_id {
    display "Access granted"
}

if not has_id {
    display "Bring an ID"
}

Advanced Pointer Usage

Use ptr handles for explicit low-level workflows without mixing old pointer syntax.

Pointer Handles Across Variables

var a @ int = 7
var b @ int = 14
var p_a @ ptr = a
var p_b @ ptr = b
display "Pointers captured"

Array Base Pointer Pattern

var data @ [int] = [5, 10, 15]
var p_data @ ptr = data[0]
display p_data

Guideline

Keep pointer-heavy code in small, testable functions and document each pointer's target clearly.

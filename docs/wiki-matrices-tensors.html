Matrices and Tensors

Matrices and tensors are central for ML and numeric computing in Linea.

Matrix Basics

var m = [[1.0, 2.0], [3.0, 4.0]]
var first_row = m[0]
display first_row

Tensor-style Nested Data

var t = [[[1, 2], [3, 4]], [[5, 6], [7, 8]]]
display t[1][0][1]

Slicing Note

Prefer explicit intermediate variables when slicing to keep transformations readable.

Comparison Operators

Comparison operators evaluate relations and return boolean values for conditionals.

Supported Operators

  • ==, !=
  • <, <=
  • >, >=
var score @ int = 87

if score >= 90 {
    display "Grade: A"
} else if score >= 80 {
    display "Grade: B"
} else {
    display "Grade: C"
}

Tip

Prefer explicit branches with comparisons over nested arithmetic expressions for readability.
